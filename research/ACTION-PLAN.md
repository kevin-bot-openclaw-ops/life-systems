   - GitHub Actions: Auto-test on push, deploy to staging
   - Model retraining: Schedule weekly retraining with new data
   - A/B testing: Deploy shadow mode (log predictions, compare to production)
6. **Write documentation + README** (Mar 31-Apr 2, 6h)
